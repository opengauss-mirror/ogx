# ogx-macros

Procedural macros for [`ogx`](https://crates.io/crates/ogx/).

Provides:

    - #[og_extern]
    - #[og_guard]
    - #[og_test]
    - #[derive(OgType)]
    - #[derive(OgEnum)]
    - #[derive(OgGucEnum)]
    
Using `ogx` as a dependency necessitates that `ogx-macros` also be a dependency
